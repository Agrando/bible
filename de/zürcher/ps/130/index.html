Language > German > Zürcher Bibel 1931 > Psalms > Chapter 130

1EIN Wallfahrtslied. Aus der Tiefe rufe ich, Herr, zu dir, / (a) Kla 3:55 2höre auf meine Stimme! / Lass deine Ohren merken auf mein lautes Flehen! / 3Wenn du die Sünden anrechnest, / Herr, wer kann bestehen? / 4Doch bei dir ist Vergebung, / auf dass man dich fürchte. / (a) Jes 55:7 5Ich hoffe auf dich, o Herr, / meine Seele hofft auf dein Wort. / 6Meine Seele harrt auf den Herrn, / mehr als die Wächter auf den Morgen. / Mehr als die Wächter auf den Morgen, / 7harre, Israel, auf den Herrn! / Denn bei dem Herrn ist die Gnade, / bei ihm ist reichlich Erlösung. / 8Ja, er wird Israel erlösen / von all seinen Sünden.
