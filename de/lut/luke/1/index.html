Darumb auch das Heilige / das von dir geboren wird / wird Gottes Son genennet werden. 36Vnd sihe / Elisabet deine gefreundete / ist auch schwanger mit einem Son / in jrem alter / vnd gehet itzt im sechsten mond / die im geschrey ist / das sie vnfruchtbar sey / 37Denn bey Gott ist kein ding vmmüglich. 38Maria aber sprach Sihe / Jch bin des HERRN magd / mir geschehe wie du gesagt hast.
