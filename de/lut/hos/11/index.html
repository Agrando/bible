1DA Jsrael jung war / hatte ich jn lieb / vnd rieff jm / meinem Son / aus Egypten. 2Aber wenn man jnen jtzt rufft / so wenden sie sich dauon / vnd opffern den Baalim / vnd reuchern den Bildern. 3Jch nam Ephraim bey seinen armen vnd leitetWie die Mutter ein Kind gengelt vnd leret gehen vnd füret bey den Armen. jn / Aber sie mercktens nicht / wie ich jnen halff. 4Jch lies sie ein menschlich Joch zihen / vnd in seelen der Liebe gehen / vnd halff jnen das Joch an jrem Hals tragen / vnd gab jnen futter / 5Das er sich ja nicht wider solt in Egyptenland keren / So ist nu Assur jr König worden / denn sie wollen sich nicht bekeren. 6Darumb sol das schwert vber jre Stedte komen / vnd sol jre Rigel auffreiben vnd fressen / vmb jres furnemens willen. 7Mein Volck ist müdeDiesen Vers sol niemand verstehen / auch im Ebreischen nicht.sich zu mir zu keren / Vnd wie man jnen predigt / so richtet sich keiner auff.
